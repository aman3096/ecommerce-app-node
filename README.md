E-commerce app built in Node.js, Express.js, and EJS(template engine) for templating
Contains the following features
1. Mongooose
2. Cookies, Sessions
3. Authentication
4. Email sending
5. Authorization
6. Error Handling with catch 
7. validation using express-validator library
8. file upload using multer
9. API pagination in mongoose
10. async requests

Database used: mongodb

Features under development
1. GraphQL
2. Websockets
